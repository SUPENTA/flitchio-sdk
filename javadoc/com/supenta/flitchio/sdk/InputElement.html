com.supenta.flitchio.sdk

Class InputElement

  • java.lang.Object
    • com.supenta.flitchio.sdk.InputElement
  • Direct Known Subclasses:
    InputElement.Button, InputElement.Joystick

    public abstract class InputElement
extends Object
    Base class describing the input elements of Flitchio: buttons and joysticks.

    Flitchio has 2 main buttons (InputElement.BUTTON_TOP and InputElement.BUTTON_BOTTOM) and 2 main joysticks (InputElement.JOYSTICK_TOP and InputElement.JOYSTICK_BOTTOM). But this class also describes additional buttons that correspond to the 4 discrete directions (UP, DOWN, LEFT, RIGHT) for each joystick. These InputElement.Button.DpadButton buttons behave exactly like normal buttons: they have a pressed/released state, and you receive events for them at the same time that you receive events for the corresponding joystick.

    The elements defined here can be used to identify the buttons and joysticks in a FlitchioSnapshot, or as source of ButtonEvents and JoystickEvents.
