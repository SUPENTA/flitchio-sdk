com.supenta.flitchio.sdk

Interface FlitchioEventListener


  • public interface FlitchioEventListener
    Listener of Flitchio events: button state changed, joystick position changed. A listener must be registered with FlitchioController.onResume(FlitchioStatusListener, FlitchioEventListener) to receive events.
    Since:
    0.6.0

      • onFlitchioButtonEvent

        void onFlitchioButtonEvent(InputElement.Button source,
                         ButtonEvent event)
        Called when the state of a button has changed (just pressed, kept pressed or just released). Check InputEvent.getAction() to determine the nature of the event.

        Note: this is called on the thread associated with the Handler specified when registering the FlitchioEventListener or an arbitrary non-UI thread if no Handler was specified.

        Parameters:
        source - The button whose state has changed.
        event - The event corresponding to the button state change.
        Since:
        0.6.0

        Note: this is called on the thread associated with the Handler specified when registering the FlitchioEventListener or an arbitrary non-UI thread if no Handler was specified.

        Parameters:
        source - The joystick whose position has changed.
        event - The event corresponding to the joystick position change.
        Since:
        0.6.0
