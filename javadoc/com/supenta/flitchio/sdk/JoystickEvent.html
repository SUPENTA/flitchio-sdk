    Object used to report joystick events.

    A joystick event is simply an update of the position of the joystick. You can retrieve the position described in this event with JoystickEvent.getX() and JoystickEvent.getY(). Joystick events don't handle the concept of press/release so the action associated with such events is always InputEvent.ACTION_MOVE.

    Each event has one source that is one of the joysticks defined in InputElement.JOYSTICKS.
