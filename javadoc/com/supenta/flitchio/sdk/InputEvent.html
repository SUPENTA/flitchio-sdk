Class InputEvent<T extends InputElement>

  • java.lang.Object
    • com.supenta.flitchio.sdk.InputEvent<T>
  • Direct Known Subclasses:
    ButtonEvent, JoystickEvent

    public abstract class InputEvent<T extends InputElement>
extends Object
    Base class for button and joystick events that are fired in listening mode.

    Each event has one source that is an InputElement and one action (InputEvent.getAction()).

    Since:
    0.5.0
